# Tongue Twister

## Introduction

Tongue Twister is a phrase that is designed to be difficult to articulate properly. The goal is to improve your speaking in any language.
This project is a personal work to train as web developper.

### Technologies

  - Node.js
  - Express
  - GraphQL
  - React.js
  - MongoDB

### Features

  - [x] Rest API
  - [x] Swagger Doc
  - [] Voice Recognition
  - [] Scoring System
  - [] Login System
